# Google Two-Factor Authenticator CLI

## Install / Configuration



## Running

% java -jar gauth_cli.jar


### Usage
```
Usage: <main class> [options]
  Options:
    -a, --account
       Use account #
       Default: -1
    -c, --config
       Config file location
       Default: ./gauth.cfg
    -h, --help
       Show usage
       Default: false
    -i, --interactive
       Display timer and codes
       Default: false
    -l, --list
       List accounts
       Default: false
    -s, --secret
       Secret
```
